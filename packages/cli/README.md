# @bunway/cli

The Bunway CLI tool for scaffolding and managing projects.

## Installation

```bash
bun add -g @bunway/cli
```
Commands
create

Create a new Bunway project:
bash

bunway create my-app

This will:

    Create a new directory with your app name
    Generate the folder structure for the selected preset
    Create necessary configuration files
    Initialize package.json and tsconfig.json

serve

(Coming in v0.2) Start the development server with hot reload.
bash

bunway serve
